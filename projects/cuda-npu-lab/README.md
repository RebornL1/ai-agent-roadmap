# cuda-npu-lab

主线项目 5：NPU 副线学习仓库。

## 项目目标

长期补齐 C++、Linux、CUDA 和 NPU Runtime 基础，作为 AI 系统软件方向的副线储备。

这个项目不是当前主线，不抢 AI Agent 前端和 AI 应用工程的时间，但用于拓展更底层 AI 岗位的可能性。

## 学习范围

- C++ 基础和工程实践
- Linux 系统编程
- CUDA 入门
- GPU/NPU 内存模型
- mini runtime
- kernel launch 概念
- 推理运行时基础

## 技术方向

- C++
- CMake
- Linux
- CUDA
- Python 辅助脚本

## MVP 验收标准

- 完成一个 C++ 命令行小工具。
- 完成一个 CUDA 向量加法 demo。
- 写清楚 CPU / GPU / NPU 执行模型差异。
- 整理一份 mini runtime 学习笔记。

## 面试可讲点

- 理解 AI 推理不只是模型服务，也涉及底层 runtime 和硬件执行。
- 能解释 host/device、kernel、memory copy、并行计算的基本概念。
- 体现跨层学习能力，但不把它包装成主线经验。
