# Day 1 Learning Log

## 今日完成

- 创建了 `projects/dba-agent-workbench` 项目。
- 初始化了 Next.js + TypeScript + Tailwind CSS 应用。
- 完成了三栏 AI Agent 工作台模拟界面：
  - 左侧：DBA 故障案例列表
  - 中间：聊天工作区
  - 右侧：Agent 执行轨迹和工具调用面板
- 只使用模拟数据，没有接真实模型。
- 补充了项目 README。
- 通过了 lint 和生产构建检查。

## 遇到的问题

- 第一次安装依赖时，npm 下载速度太慢，导致安装超时。
- pnpm 11 默认拦截了部分原生依赖的构建脚本。
- 通过在 `pnpm-workspace.yaml` 中允许 `sharp` 和 `unrs-resolver` 构建，解决了安装问题。

- 这个项目把真实 DBA 故障处理流程转成了 AI Agent 产品界面。

- 从 Day 1 开始就把只读诊断和人工审批状态展示出来，体现生产安全意识。
- 这是后续接入 RAG、MCP、LangGraph 和 Human Approval 的前端基础。
